# Enfield Dev Ltd — Recruitment Website

A static recruitment website for Enfield Dev Ltd, a fictional web development company based in Enfield, London. Built as part of a BTEC Level 3 Web Development assignment.

## Pages

| File | Page |
|---|---|
| index.html | Home |
| team.html | Meet the Team |
| job-role.html | Job Roles |
| apply.html | Apply |
| courses.html | Courses |
| location.html | Location |
| support.html | Support |
| about-contact.html | About & Contact |

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks or libraries)

## How to Open

Open any `.html` file directly in a web browser. No server or build step is required.
